# Jalal Insurance Broker Pricing Tool

Public static insurance pricing website for Jalal Insurance Broker.

## Files to upload to GitHub repository root

- index.html
- styles.css
- script.js
- README.md
- UPLOAD_GUIDE.txt
- .nojekyll
- assets/
  - jalal-profile-cutout.png
  - blinq-business-card-qr.png

## Notes

The website is static. It stores no submitted client data. Quote requests are sent through WhatsApp using prefilled messages.

The Motor calculator uses the corrected Motor Tariff V17 effective June 19, 2026.
